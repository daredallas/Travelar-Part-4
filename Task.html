Tasks



Task Description

You just had a great meal from a restaurant that you stumbled upon in a foreign country. Your experience was so great and you would like to leave a review for the restaurant to serve as a memory of your trip. You come to the same place the next day but it is packed so you would like to find a similarly priced restaurant nearby:

1. Leave a review
2. Find an alternative, similarly priced restaurant (additional task)


Cognitive Walktrhough

  1. First step, the user can see the login/signup page

  2. User taps on login

  3. User sees two fields, username and password + a “Log In” button

  4. User enters credentials and taps on log in

  5. User is taken to an AR view

    • User thinks - “I see an image through my camera, I want to find a restaurant so that it can tell me what it is”

    • Recommended action - move the camera to point at the restaurant

  6. Place name appears on the screen where the restaurant is

    • User thinks - “I can see the restaurant and its name, I want to leave a review now

    • Recommended action - tap on the bubble with the restaurant’s name

  7. User sees a bubble with some info about the restaurant and a ”more” button

    • User thinks - “I see no review option, but there is a description and more button”

    • Action - “Click on the more button”

  8. User sees the full restaurant page, with a lot of info, review button at the top

    • User thinks - “Ok, there’s some more functionality here, I’m still looking for a review option”

    • Recommended action - The review button is at the top, click it and it takes you to the review page

  9. User is taken to the Review page:

    • User thinks - “I see text field and a line of empty stars”

    • Recommended action - Fill out text field and stars
